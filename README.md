# 분별깔때기 시뮬레이션 — 밀도 차를 이용한 분리

중학교 과학 `[9과08-03]` 물질의 특성(밀도)을 이용한 혼합물 분리 학습용 웹 시뮬레이션.
물(1.00 g/mL)과 식용유(0.92 g/mL)의 **밀도 차**로 두 액체를 분리하는 과정을 단계별로 조작해 본다.

## 실행

정적 HTML 한 파일. 빌드·의존성 없음.

- 로컬: `index.html`을 브라우저로 열기
- 배포: Vercel 정적 호스팅 (루트의 `index.html`이 진입점)

## 사용법

1. **정치하기** — 섞인 혼합물이 두 층(위=식용유, 아래=물)으로 분리될 때까지 기다림
2. **마개 열기 → 꼭지 열기** — 아래층(물)을 비커 (가)에 받음 (다 받으면 꼭지 자동 잠금)
3. **위층 따라내기** — 남은 식용유를 입구로 비커 (나)에 옮김 → 분리 완료

의도적 실수도 학습 가능: 정치 전 꼭지 열기 / 마개 없이 꼭지만 열기 → 경고 안내.

## 파일

- `index.html` — 시뮬레이션 (배포 진입점)
- `separatory-funnel-sim-v1.0.html` — 원본 (index.html과 동일 내용)
